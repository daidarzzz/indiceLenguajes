1.- ¿Cuántos H1 puede o debe de tener una página HTML?

Como tal, una página puede tener los H1 que quiera, pero, lo recomendable es que se utilice solo un H1 para el título.


2.- ¿Cómo se está definiendo el fondo de la página? ¿Es correcto? Estas respuestas y la de los siguientes ejercicios las metes en el body de los html.

Se está usando un "style = bgcolor" como atributo del body. Esto no es la forma correcta de hacerlo, la mejor manera sería utilizando un background-color dentro de un archivo css.
